# Weather and Energy Consumption – Data Wrangling Project

This repository contains a data wrangling project analyzing the relationship between weather and energy consumption.
The project gathers weather data via the NOAA API and combines it with energy consumption data from the U.S. Energy Information Administration.

The analysis, cleaning steps, and visualizations are fully contained in the Jupyter notebook:
`weather_energy_wrangling.ipynb`.

All raw and cleaned datasets used in the analysis are included in the `data/` directory.
